# 이 문서를 읽기 전...
이 파트는 Backend 파트입니다. Frontend에게 전달해야 하는건 맞고, Frontend적인 요소를 넣으시면 안돼요.

# 회원가입 순서
1. NAVER / Kakao / Google / Apple 계정 중 당사 DB에 없는 계정을 연동
2. 연동 완료 시 첫 연동에 대해 들고온 정보가 있으면 이걸 그대로 넣어서 가입은 완료(없어도 일단 가입은 처리함)
3. 이메일, 닉네임 정보부터 맞는지 확인하는 페이지 띄움.(이메일 값 가져온거 없으면 null로 들고 오겠죠?) -> 이 두 정보는 필수정보임. 그리고 프사.
프사는 없으면 null 해도 됨. 이 상태로 다음 누르면 POST 통신으로 저장함.
4. 프론트쪽에서 [추가] 개인정보수집 동의(True)를 한 상태로(만약, 소셜 연동 단계에서 바로 동의를 했다면? 바로 들고옴) 생년월일(소셜에서 수집됐으면 입력되어 있을거), 성별(소셜에서 수집됐으면 입력되어 있을거), 그리고 기능명세서상 더 들고와야 하는거 하고 가입 완료!
